World Population Project-R programming

World Population Project is a project that contains up-to-date datasets on the population of all the countries across the world in different years. The datasets also contain the density and area covered by each country. During the project, data cleaning and analysis were conducted using R Programming packages such as tidy verse, dplyr, shinny, and tidyr and visualized using ggplot. Some of the analysis was the ranking of countries and the continent by population. Growth patterns of the population were also identified in different countries. The area of the countries was identified, and it was seen how it varies with the population. Finally, data was visualized using ggplot2, and more insights were derived.

World Population Project-SQL

The project is an extension of the previous project performed using R programming language. Excel and SQL tools were used, and data was cleaned using an Excel sheet. SQL was used to analyze data and derive insights from the data. The project aimed to derive insights into population growth across the years, population distribution in each country and Continent, and the relationship between population and the area covered by the country. Some of the insights derived is that there is a correlation between population and area where it was evident that there was a high population in countries with large areas. The project also showed that the population and area of a country determine the population density of a country.
